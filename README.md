# ✨ LUXE — Premium E-Commerce Boutique

> **Luxe** is a full-stack e-commerce platform designed with a focus on luxury aesthetic and high-end functional experience. Built for the modern shopper, it delivers a smooth, glassmorphic UI alongside a powerful administrative system for inventory and order management.

![Preview](https://images.unsplash.com/photo-1556821840-3a63f95609a7?w=1200&h=400&fit=crop)

---

## 🌟 Key Features

### 🛍️ Client Experience
*   **Premium UI/UX**: Elegant "Emerald & Ivory" palette with glassmorphism for a high-end boutique feel.
*   **Dynamic Product Discovery**: Browse by category (Clothing, Shoes, Accessories, New Arrivals) with real-time filtering and search.
*   **Smooth Navigation**: Integrated animations with `framer-motion` for fluid page transitions.
*   **Interactive Shopping**: Quick View modals, wishlist support, and a dedicated shopping cart sidebar.
*   **Product Details**: Comprehensive depth including multiple sizes, colors, and interactive size charts.

### 🛡️ Admin Portal (`/admin`)
*   **Inventory Management**: Full CRUD (Create, Read, Update, Delete) dashboard for product listings.
*   **Interactive Dashboard**: Real-time sales statistics, revenue tracking, and order/user counts.
*   **Order Management**: Track customer orders and update shipping statuses.
*   **Customer & VIP Hub**: Manage user profiles and premium "VIP List" subscribers.

## 🚀 Tech Stack

### Frontend
*   **React 18** (Vite)
*   **React Router DOM v6** (Navigation)
*   **Axios** (API Requests)
*   **Framer Motion** (Animations)
*   **React Icons** (Feather Icons/fi)
*   **Vanilla CSS** (Component-scoped modules)

### Backend
*   **Node.js / Express.js**
*   **JSON Database** (Flat-file architecture for lightweight performance)
*   **JWT (JSON Web Token)** (Secure Authentication)
*   **Multer** (Automated Product Image Uploads)

## 🛠️ Quick Start

### 1. Prerequisites
*   Node.js installed
*   NPM installed

### 2. Setup
Clone the repository and install dependencies in both `frontend` and `backend` directories.

```bash
# Backend setup
cd backend
npm install
npm start

# Frontend setup
cd frontend
npm install
npm run dev
```
